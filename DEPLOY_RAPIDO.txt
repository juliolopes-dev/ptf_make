PASSO 1: GITHUB (3 min)
─────────────────────────────────────────────────────────

1. Criar conta: https://github.com (se não tiver)

2. Criar repositório: https://github.com/new
   Nome: ptf-maker
   Público ou Privado: Qualquer um
   
3. No PowerShell (pasta do projeto):

   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/SEU_USUARIO/ptf-maker.git
   git push -u origin main

─────────────────────────────────────────────────────────
PASSO 2: STREAMLIT CLOUD (5 min)
─────────────────────────────────────────────────────────

1. Acessar: https://share.streamlit.io

2. Login com GitHub

3. Click "New app"

4. Preencher:
   Repository: ptf-maker
   Branch: main
   File: app.py

5. Advanced settings → Secrets:

   PEXELS_KEY = "SUA_CHAVE_AQUI"

6. Click "Deploy!"
